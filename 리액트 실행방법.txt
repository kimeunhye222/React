💕 React project 생성하는 방법 
1. 리액트 프로젝트를 생성하고자 하는 폴더로 vscode 를 킨다.
2. npm create vite@latest 프로젝트이름 -- template React
      * 주의사항 : -- template 사이를 띄우면 안된다.
3. Use ~> => No
   - 새버전 쓸래? 이런 이야기임
4. 설치하고 바로 실행할래? => yes

👌 React project를 실행하는 방법
- 나의 상황에 따라 다르게 행동

case1. 서버가 켜져있다. (터미널에 값을 입력할 수 없는 상태)
   - 1-1. 만약 켜져있는 서버가 본인이 희망하는 프로젝트라면 그대로 실행
        => localhost: 포트번호로 이동하면 됨

    1-2. 그게 아니라면? Ctrl+C 연타해서 서버 꺼주기
    1-3. cd + 경로 (내가 키고싶은 프로젝트)
    1-4. npm run dev

case2. 내 터미널의 마지막 경로가 React 다 (== 내 모드모듈이 있는 위치가 아니다.)
1-3, 1-4 진행

case3. 내 터미널의 마지막 경로가 project이다( == 내 노드모듈이 있는 위치다)
1-4 진행
